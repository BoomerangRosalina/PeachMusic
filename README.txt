PEACH MUSIC SOUND BOARD SOURCE CODE
===========================================
Here is the orignal Peach Music Source code.

This source code's purpose is to be able to play music files in a simple way within a simple play command

A sample audio song will be provided for demonstration.

===========================================
SETUP INSTRUCTIONS/HOW TO USE

REQUIRED Packages
- discord.py
- ffmpeg
- PyNaCl
- discord-py-slash-command (Optional if you are using main.py. Generally requires Python 3.8.6)

Instructions for Setup
1. Download this Source code
2. Copy main.py to your hosting provider/computer (You may also need the requirements.txt file as well)
3. Upload the Audio files you wish to use for your soundboard bot. (This soundboard bot source code is meant for .mp3 files)
4. Install the needed packages above. (discord-py-slash-command is not needed if you are using mainnoslash.py)
5. At the bottom with client.run("INSERTBOTTOKENHERE"), replace INSERTBOTTOKENHERE with your bot token.
6. Save Everything and run the bot.

HOW TO USE
Lets say we want to play marioburned.mp3 (provided mp3 file)
1. Invite your bot if you have not already done so
2. Join a Voice Channel
3. Use [prefixhere]connect
4. Use [prefixhere]play marioburned
That should play your audio file.

NOTE: PERFORMANCE MAY BE DIFFERENT DEPENDING ON BOT HOSTING SERVICE USED
